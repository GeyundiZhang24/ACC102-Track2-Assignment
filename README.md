# ACC102 Track2 Assignment: SBA Loan Data Analysis (2023-2025)
A Python data analysis project for ACC102 Mini Assignment Track2.

## Project Overview
This project analyzes SBA loan data from 2023 to 2025 to explore loan trends, revenue patterns, and small business financing characteristics.

## File Structure
- ACC102_Analysis.ipynb: Main Python analysis notebook
- sba_loan_filtered_2023-2025.csv: Dataset used in this project
- revenue_trend.png: Visualization output
- README.md: Project introduction

## Tools & Libraries
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Clear revenue trends across 2023-2025
- Stable distribution of SBA loan support
- Identifiable periodic fluctuations
